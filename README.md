# 📊 GA4 E-commerce Funnel Analysis

This project analyzes user behavior across an e-commerce funnel using GA4 BigQuery data.

## 🎯 Objective
Identify drop-off points and improve conversion rate.

## 🧠 Data Source
- GA4 (Google Analytics 4)
- BigQuery Public Dataset

## ⚙️ Tools
- SQL (BigQuery)
- Looker Studio

## 🔍 Funnel Steps
- Session
- View Item
- Add to Cart
- Checkout
- Purchase

## 📈 Key Metrics
- 355K Sessions
- 5.5K Purchases
- 1.57% Conversion Rate

## 💡 Insights
- Major drop-off between view item and add to cart
- Low conversion rate
- High traffic but low quality
- Mobile performance is weak

## 🚀 Recommendations
- Improve product page UX
- Optimize checkout process
- Improve mobile experience
- Run A/B tests
